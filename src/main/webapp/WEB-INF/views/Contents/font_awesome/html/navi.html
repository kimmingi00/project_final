그룹만들기

그룹을 만드는 태그에는 section, article, div태그들이 있다.

태그

  • 모든 태그는 자신만의 스타일을 가지고 있다
  • 순서가 없는 목록 태그는 UL태그이다

하나의 문단을 나타내는 태그는 p태그이다

스타일이란, 글자크기, 글자색상, 밑줄, 바깥여백, 안쪽여백, 제목 등 여러가지의 속성들을 스타일이라고 한다.

스타일을 내가 수정하는 방법

내부 스타일 시트, 외부 스타일 시트, 태그에 직접 style 속성을 부여하는 방법이 있다

스타일 시트를 부여하는 방법에는 태그, class(.), id(#)을 이용하는 방법이 있다

하위 선택자를 이용하여 스타일 시트를 변경하는 방법도 있다
